Django Setup Commands for Ubuntu/Linux

1. Update and Install Python & Pip
sudo apt update && sudo apt install python3 python3-pip python3-venv -y
Copied!
2. Create Virtual Environment
python3 -m venv myenv
Copied!
3. Activate Virtual Environment
source myenv/bin/activate
Copied!
4. Install Django
pip install django
Copied!
5. Create Django Project
django-admin startproject myproject
Copied!
6. Navigate into Project Folder
cd myproject
Copied!
7. Start New App
python3 manage.py startapp myapp
Copied!
8. Make Migrations
python3 manage.py makemigrations
Copied!
9. Apply Migrations
python3 manage.py migrate
Copied!
10. Create Superuser
python3 manage.py createsuperuser
Copied!
11. Run Development Server
python3 manage.py runserver
Copied!
12. Collect Static Files
python3 manage.py collectstatic
Copied!
13. Freeze Installed Packages
pip freeze > requirements.txt
Copied!
14. Install Gunicorn (Production)
pip install gunicorn
Copied!
15. Run Gunicorn Server
gunicorn --bind 0.0.0.0:8000 myproject.wsgi
Copied!
16. Install WhiteNoise for Static Files
pip install whitenoise
Copied!
17. Install Useful Django Packages
pip install djangorestframework pillow psycopg2-binary
Copied!
18. Deactivate Virtual Environment
deactivate
Copied!
19. Run Server on Public IP
python3 manage.py runserver 0.0.0.0:8000
Copied!
20. Install All from requirements.txt
pip install -r requirements.txt
Copied!
21. Update and Install Essentials
sudo apt update && sudo apt install python3 python3-venv python3-pip -y
Copied!
22. Create and Activate Virtual Environment
python3 -m venv myenv && source myenv/bin/activate
Copied!
23. Install Django & Gunicorn
pip install django gunicorn
Copied!
24. Start Project and App
django-admin startproject myproject && cd myproject && python3 manage.py startapp myapp
Copied!
25. Migrate and Create Superuser
python3 manage.py migrate && python3 manage.py createsuperuser
Copied!
6. Test Gunicorn Server
gunicorn --bind 0.0.0.0:8000 myproject.wsgi
Copied!
27. Create Gunicorn Service
sudo nano /etc/systemd/system/gunicorn.service
Copied!
[Unit]
Description=gunicorn daemon
After=network.target

[Service]
User=ubuntu
Group=www-data
WorkingDirectory=/home/ubuntu/myproject
ExecStart=/home/ubuntu/myenv/bin/gunicorn --workers 3 --bind unix:/home/ubuntu/myproject/gunicorn.sock myproject.wsgi:application

[Install]
WantedBy=multi-user.target
28. Enable and Start Gunicorn
sudo systemctl start gunicorn && sudo systemctl enable gunicorn && sudo systemctl status gunicorn
Copied!
29. Install and Configure Nginx
sudo apt install nginx -y
Copied!
30. Create Nginx Config
sudo nano /etc/nginx/sites-available/myproject
Copied!
server {
    listen 80;
    server_name example.com www.example.com;

    location = /favicon.ico { access_log off; log_not_found off; }
    location /static/ {
        root /home/ubuntu/myproject;
    }

    location / {
        include proxy_params;
        proxy_pass http://unix:/home/ubuntu/myproject/gunicorn.sock;
    }
}
31. Enable Nginx Site
sudo ln -s /etc/nginx/sites-available/myproject /etc/nginx/sites-enabled && sudo nginx -t && sudo systemctl restart nginx
Copied!
32. Allow Nginx Through Firewall
sudo ufw allow 'Nginx Full' && sudo ufw enable && sudo ufw status
Copied!
33. Install Cloudflare Tunnel (cloudflared)
sudo apt install cloudflared -y
Copied!
34. Login Cloudflare Tunnel
cloudflared tunnel login
Copied!
35. Create Tunnel and Run
cloudflared tunnel create mytunnel
Copied!
36. Configure Tunnel to Django Server
sudo nano ~/.cloudflared/config.yml
Copied!
tunnel: mytunnel
credentials-file: /home/ubuntu/.cloudflared/mytunnel.json

ingress:
  - hostname: yourdomain.com
    service: http://localhost:8000
  - service: http_status:404
37. Run Cloudflare Tunnel
cloudflared tunnel run mytunnel
Copied!
38. Run on Background (Service)
sudo systemctl enable cloudflared && sudo systemctl start cloudflared
Copied!
39. Check Everything
sudo systemctl status gunicorn nginx cloudflared
Copied!
40. Collect Static Files for Deployment
python3 manage.py collectstatic
Copied!
